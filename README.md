# ARTI308 Lab 5 – Student Performance Analysis

This project applies machine learning and data analysis techniques to a student performance dataset. The goal of the notebook is to explore the data, visualize important patterns, and build a predictive model that can classify whether a student will pass or fail.

The dataset used in this lab contains information about students such as study time, absences, alcohol consumption, family background, and grades. By analyzing these features, we attempt to understand which factors have the strongest impact on the final grade.

The notebook includes several steps such as data loading, data inspection, visualization, feature engineering, preprocessing, model training, and model evaluation. A Random Forest classifier is used to predict student performance, and the results are evaluated using accuracy, classification reports, and confusion matrices.

This project demonstrates how machine learning techniques can be used to analyze educational data and potentially help identify students who may need additional academic support.
